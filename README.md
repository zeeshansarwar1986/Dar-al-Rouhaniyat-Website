# Dar al-Rouhaniyat (دار الروحانیت) 🕌

A premium, spiritual, and e-commerce platform dedicated to spiritual wellness, gemstones, and charitable work. This project is built using modern web technologies to provide a seamless, respectful, and high-performance experience.

---

## 🌟 Overview

**Dar al-Rouhaniyat** is a bespoke web application designed for the modern spiritual practitioner. Inspired by the aesthetic of `zanjani.pk`, it combines deep cultural heritage with state-of-the-art e-commerce functionality.

### Key Features
- **Premium Design**: A sophisticated Deep Red and Gold theme with authentic Urdu typography (Noto Nastaliq Urdu).
- **Gemstone Catalog**: A fully responsive product grid with categorization and detailed filtering.
- **Cart System**: A sleek, side-drawer cart for quick additions and a smooth checkout flow.
- **Charity (مدد) Page**: A dedicated section for community support, including ration programs and educational scholarships.
- **RTL Support**: Built from the ground up to support Urdu and other right-to-left languages.
- **Interactive Assets**: High-definition, AI-curated spiritual wallpapers and responsive UI elements.

## 🛠 Developer Guide

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/zeeshansarwar1986/Dar-al-Rouhaniyat-Website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd dar-al-rouhaniyat
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```

### Building for Production
Create an optimized production build:
```bash
npm run build
```

## 📝 Customization Reference

### Changing the Hero Image
The hero image is located in `public/images/karbala_night.png`. To update it, simply replace this file or update the URL in `src/components/Hero.css`.

### Adding Products
Products are currently managed in `src/pages/GemsPage.jsx`. Update the `products` array to add, remove, or modify gemstones.

### Editing the Help Page
The Help page content is in `src/pages/HelpPage.jsx`. You can easily adjust the text and programs offered.
